## 🚀 Deploy no GitHub Pages

### Passo 1: Criar repositório
1. Acesse [github.com](https://github.com) e faça login
2. Clique em **"New repository"**
3. Nome: `smart-wallet` (ou outro de sua preferência)
4. Marque como **Public** (obrigatório para GitHub Pages gratuito)
5. Clique em **"Create repository"**

### Passo 2: Fazer upload dos arquivos
1. Dentro do repositório, clique em **"Add file" → "Upload files"**
2. Arraste os seguintes arquivos:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `favicon.svg`
   - `README.md`
3. Clique em **"Commit changes"**

### Passo 3: Ativar GitHub Pages
1. Vá em **Settings** (configurações do repositório)
2. No menu lateral, clique em **Pages**
3. Em **"Source"**, selecione:
   - Branch: **main**
   - Pasta: **/ (root)**
4. Clique em **Save**
5. Aguarde ~1 minuto

### Passo 4: Acessar seu WebApp
Sua aplicação estará disponível em:
```
https://rogerelizar.github.io/SmartWallet-2026/
```

## 📱 Instalar como App

### No Android (Chrome):
1. Acesse a URL no Chrome
2. Toque no menu (⋮) → **"Instalar aplicativo"** ou **"Adicionar à tela inicial"**
3. Confirme a instalação

### No iPhone (Safari):
1. Acesse a URL no Safari
2. Toque no botão **Compartilhar** (quadrado com seta)
3. Role e toque em **"Adicionar à Tela de Início"**
4. Confirme

### No Desktop (Chrome/Edge):
1. Acesse a URL
2. Clique no ícone de instalação na barra de endereços (⊕)
3. Confirme a instalação

## ✨ Funcionalidades

- ✅ Controle completo de receitas e despesas
- ✅ Categorias personalizáveis com cores

## 🔒 Privacidade

Todos os dados são armazenados **localmente no seu navegador** via `localStorage`. Nenhuma informação é enviada para servidores externos.
